# 二、安装

# 1. 安装

命令行运行如下命令，全局安装 TypeScript：

npm install -g typescript

安装完成后，在控制台运行如下命令，检查安装是否成功：

tsc -V

# 2. 执行

安装完成后可以使用 tsc 命令来执行 TypeScript 的相关代码：

  1. 新建一个 app.ts 的文件：

    var message:string = "Hello World" 
console.log(message)

  2. 执行 tsc 命令将 TypeScript 转换为 JavaScript 代码：

    tsc app.ts

  3. 此时在当前目录下（与 app.ts 同一目录）就会生成一个 app.js 文件，代码如下：

    var message = "Hello World";
console.log(message);

  4. 使用 node 命令来执行 app.js 文件：

    node app.js 
打印结果：Hello World
