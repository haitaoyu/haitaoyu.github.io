# 八、其他

# 1. 声明文件

TypeScript 作为 JavaScript 的超集，在开发过程中不可避免要引用其他第三方的 JavaScript 的库。虽然通过直接引用可以调用库的类和方法，但是却无法使用TypeScript 诸如类型检查等特性功能。

通过引用相关第三方库的声明文件，就可以借用 TypeScript 的各种特性来使用库文件了。

推荐使用 @types 统一管理第三方库的声明文件。@types 的使用方式很简单，直接用 npm 安装对应的声明模块即可（戳这 (opens new window)搜索你需要的声明文件），以 jQuery 为例：

npm install @types/jquery --save-dev

然后需要把声明语句放到一个单独的文件 jQuery.d.ts 中，这样就可以正常的在 TypeScript 文件中使用 jquery 了。

// src/jQuery.d.ts
// 全局变量这种模式的声明文件
declare var jQuery: (selector: string) => any;

# 2. 内置对象

JavaScript 中有很多内置对象，它们可以直接在 TypeScript 中当做定义好的类型。

# 1) ECMAScript 的内置对象

  • Boolean
  • Number
  • String
  • Date
  • RegExp
  • Error
let b: Boolean = new Boolean(1)
let n: Number = new Number(true)
let s: String = new String('abc')
let d: Date = new Date()
let r: RegExp = /^1/
let e: Error = new Error('error message')
b = true

# 2) BOM 和 DOM 的内置对象

  • Window
  • Document
  • HTMLElement
  • DocumentFragment
  • Event
  • NodeList
const div: HTMLElement = document.getElementById('test')
const divs: NodeList = document.querySelectorAll('div')
document.addEventListener('click', (event: MouseEvent) => {
  console.dir(event.target)
})
const fragment: DocumentFragment = document.createDocumentFragment()
