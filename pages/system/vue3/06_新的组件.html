# 六、新的组件

# 1. Fragment

  • 在 Vue2 中，组件必须有一个根标签。
  • 在 Vue3 中，组件可以没有根标签，内部会将多个标签包含在一个 Fragment 虚拟元素中。
  • 好处是可以减少标签层级，减小内存占用。

# 2. Teleport

Vue3 中的组件模板属于该组件，如果想把模板的内容移动到当前组件之外的 DOM 中，就可以使用 Teleport。

一个常见的场景是组件中点击按钮弹出模态框。

大多数情况下，我们希望模态框的逻辑存在于组件中，但这样模态框的位置（在整个应用中间）就很难通过 CSS 来解决，一般是使用position: fixed等 CSS 属性强行把它定位到了应用的中间位置，属于没有办法的办法。

使用 Teleport 可以解决上述问题，使用 to 这个属性可以把组件渲染到想要的位置：

<template>
  <teleport to="body">
    <div class="modal-bg" v-if="visible">
      <div class="modal-content">
        <button class="close" @click="$emit('close-modal')"> X </button>
        <div class="model-title">{{title}}</div>
        <div class="model-body">
          <slot>
            模态对话框
          </slot>
        </div>
      </div>
    </div>
  </teleport>
</template>
