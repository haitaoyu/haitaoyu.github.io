# Ajax

# 1. 简介

Ajax 是一种在无需重新加载整个网页的情况下，能够更新部分网页的技术。也就是所谓的异步加载。

它的基本工作流程如下：

使用起来也非常简单：

<head>
  <script>
    function loadXMLDoc () {
      // Ajax 脚本执行 
    }
  </script>
</head>

<div id="myDiv">
  <h2>使用 Ajax 修改该文本内容</h2>
</div>
<button type="button" onclick="loadXMLDoc()">修改内容</button>

这样就能实现网页的异步数据加载，重点在于 Ajax 脚本执行部分，下面详细说明一下创建一个 Ajax 脚本的具体步骤。


# 2. 使用步骤

创建一个 Ajax 脚本分为 4 个步骤：

# 1）创建 XMLHttpRequest 对象

不同的浏览器使用的异步调用对象有所不同：

  • 所有现代浏览器（Chrome、Firefox、Safari、IE7+ 以及 Opera）使用 XMLHttpRequest 对象：

    const xmlHttp = new XMLHttpRequest()

  • 老版本的 Internet Explorer（IE5 和 IE6）使用 ActiveXObject 对象：

    const xmlHttp = new ActiveXObject('Microsoft.XMLHTTP')

为了兼容所有浏览器，使用 Ajax 前需要检查浏览器是否支持 XMLHttpRequest 对象。如果支持，则创建 XMLHttpRequest 对象，如果不支持，则创建 ActiveXObject ：

let xmlHttp = ''

if (window.XMLHttpRequest) {
  xmlHttp = new XMLHttpRequest()
} else {
  xmlHttp = new ActiveXObject('Microsoft.XMLHTTP')
}

# 2）向服务器发送请求

使用 XMLHttpRequest 对象的 open() 和 send() 方法，将请求发送到服务器：

xmlHttp.open('GET', 'test.txt', true)
xmlHttp.send()

  • open(method, url, async)：规定请求的类型、URL 以及是否异步处理请求。

    • method：请求的类型，GET 或 POST。
    • url：文件在服务器上的位置。
    • async：true（异步）或 false（同步）。

  • send(string)：将请求发送到服务器。

    • string：仅用于 POST 请求。

# 3）获取服务器响应

使用 XMLHttpRequest 对象的 responseText 或 responseXML 属性来获取来自服务器的响应。

  • responseText：获取字符串形式的响应数据。

    document.getElementById("myDiv").innerHTML = xmlhttp.responseText

  • responseXML：获取 XML 形式的响应数据。

    const xmlDoc = xmlhttp.responseXML
let txt = ''

// 解析 XML 形式的数据
const x = xmlDoc.getElementsByTagName('ARTIST')
for (let i = 0; i < x.length; i++) {
  txt = txt + x[i].childNodes[0].nodeValue + '<br/>'
}
document.getElementById('myDiv').innerHTML = txt

# 4）完善 Ajax 请求

经过以上 3 步，已经实现了一个完整的 Ajax 请求过程。但是整个过程中还存在一个问题：无法得知服务器响应是否成功。

如果响应未成功，那么后面的操作就毫无效果，为了防止这种情况，可以监听一下响应的状态，确保响应成功后才进行后面的操作。

判断响应是否成功，需要先了解 XMLHttpRequest 对象的 3 个重要属性：

  • readyState： XMLHttpRequest 对象的状态。

    XMLHttpRequest 对象从创建开始，到发送数据、接收数据，它一共会经历以下 5 种状态：

    • 0：请求未初始化。

    • 1：服务器连接已建立。

    • 2：请求已接收。

    • 3：请求处理中。

    • 4：请求已完成，且响应已就绪。

  • onreadystatechange 事件：readyState 属性改变时的回调。

  • status： HTTP 响应状态码，如 200、400、500、404 等。


当 readyState 等于 4 且 status 为 200 时，表示响应已就绪：

xmlHttp.onreadystatechange = function () {
  if (xmlHttp.readyState === 4 && xmlHttp.status === 200) {
    document.getElementById('myDiv').innerHTML = xmlHttp.responseText
  }
}

# 3. 完整示例

// 1.创建 XMLHttpRequest 对象
let xmlHttp = ''
if (window.XMLHttpRequest) {
  xmlHttp = new XMLHttpRequest()
} else {
  xmlHttp = new ActiveXObject('Microsoft.XMLHTTP')
}

// 2.发送服务器请求
xmlHttp.open('GET', 'test.txt', true)
xmlHttp.send()

// 3.判断服务器响应是否成功，如果响应成功，进行相关操作
xmlHttp.onreadystatechange = function () {
  if (xmlHttp.readyState === 4 && xmlHttp.status === 200) {
    document.getElementById('myDiv').innerHTML = xmlHttp.responseText
  }
}
